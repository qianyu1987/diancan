# 🤖 AI 圆桌讨论 - 自定义配置指南

## 📝 快速开始

### 1. 打开应用文件
```
D:\aiwork\ai_team_roundtable.py
```

### 2. 找到 AI 配置部分
在文件的最上面，你会看到：

```python
CUSTOM_TEAM = {
    1: {
        "name": "🎯 豆包",
        "role": "全局军师",
        "color": "#ff4757",
        "think": "我在深度分析话题核心逻辑...",
        "speak": "我认为核心是效率与公平的最优平衡！"
    },
    # ... 其他 AI 配置
}
```

### 3. 修改 AI 信息
编辑每个 AI 的信息：
- `name`: AI 的名称（包括 emoji）
- `role`: AI 的角色/职责
- `color`: AI 的颜色（十六进制代码）
- `think`: AI 思考的内容
- `speak`: AI 发言的内容

### 4. 保存并重启
保存文件，然后重新启动应用即可看到新的配置！

## 💡 配置建议

### 命名规范
- 使用 emoji + 名称
- 例如：🚀 架构师、💻 工程师、🎨 设计师

### 角色描述
- 简洁明了（不超过10个字）
- 体现专长
- 例如：系统设计、代码工程、产品策略

### 颜色选择
- 选择对比度高的颜色
- 避免相似的颜色
- 参考上面的颜色表

### 思考和发言
- 思考：AI 在思考什么（不超过30个字）
- 发言：AI 的观点（不超过50个字）
- 保持简洁有力

## ❓ 常见问题

### Q: 修改后没有生效？
A: 需要重新启动应用。关闭浏览器，重新运行 `start.bat`。

### Q: 如何添加更多 AI？
A: 目前支持 6 个 AI。如果需要更多，需要修改 HTML 代码。

### Q: 颜色代码怎么找？
A: 可以使用在线颜色选择器，例如 https://www.color-hex.com/

### Q: 如何恢复默认配置？
A: 删除你的修改，恢复原始配置即可。
