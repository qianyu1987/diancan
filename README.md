# 🤖 AI 6人圆桌讨论大会

## 📋 项目概述

这是一个**AI 6人专家团队讨论模拟器**，用于订餐小程序（DingCan）的开发讨论。

6个 AI 模型围坐成圆形，轮流发言讨论技术方案，生成的代码自动保存到项目。

## 📝 使用流程

### 1. 启动应用
```bash
双击 start.bat 或 start_ai_team_roundtable.bat
```

### 2. 打开浏览器
```
http://localhost:8501
```

### 3. 输入讨论主题
例如：`开发菜单浏览功能，支持分类筛选`

### 4. 点击"开始讨论"
初始化讨论

### 5. 点击"下一轮"或直接点击小人
下一个 AI 发言

### 6. 观看讨论
- 发言时小人站起来、放大、跳跃、发光
- 底部对话框显示完整内容
- 显示思考过程和代码块

### 7. 保存代码
点击"💾 保存代码"按钮

### 8. 讨论完成
所有代码自动保存到 `D:\dingcan` 项目

## 💡 使用技巧

### 1. 获得更好的代码
- 提供详细的讨论主题
- 增加讨论轮次（8-10轮）
- 让模型充分讨论和优化

### 2. 快速保存代码
- 点击代码块下的"💾 保存代码"按钮
- 或等待讨论完成后自动保存

### 3. 查看讨论分析
- 讨论完成后显示统计信息
- 包括发言数、字数、代码块数等
- 显示发言者排行

### 4. 充分利用 Claude 优化师
- 设置足够的讨论轮次
- 让前5轮充分讨论
- Claude 会在最后一轮进行整合和优化

## 📞 常见问题

### Q: 应用启动很慢？
A: 首次启动需要加载模型，请耐心等待。

### Q: 代码保存到哪里了？
A: 所有代码保存到 `D:\dingcan` 项目目录。

### Q: 如何修改讨论轮次？
A: 在应用中使用滑块调整轮次（3-6轮）。

### Q: 如何查看完整的讨论内容？
A: 点击对话记录中的"展开"按钮查看完整内容。

### Q: 如何重新开始讨论？
A: 点击"🗑️ 清空"按钮清空讨论，然后重新开始。
