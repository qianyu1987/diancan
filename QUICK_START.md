# 🚀 快速参考

## 启动应用

### 推荐方式
```bash
双击 start.bat
选择 1 (圆桌讨论版本)
```

### 或直接启动
```bash
双击 start_ai_team_roundtable.bat
```

### 或命令行
```bash
python -m streamlit run ai_team_roundtable.py
```

```
http://localhost:8501
```

## 使用步骤

1. 输入讨论主题
2. 点击"开始讨论"
3. 点击"下一轮"或直接点击小人
4. 观看讨论和代码生成
5. 点击"💾 保存代码"保存
6. 讨论完成后自动保存所有代码

## 常见问题

**Q: 应用启动很慢？**
A: 首次启动需要加载，请耐心等待。

**Q: 代码保存到哪里？**
A: 保存到 `D:\dingcan` 项目目录。

**Q: 如何修改讨论轮次？**
A: 在应用中使用滑块调整（3-6轮）。

**Q: 如何重新开始？**
A: 点击"🗑️ 清空"按钮。
